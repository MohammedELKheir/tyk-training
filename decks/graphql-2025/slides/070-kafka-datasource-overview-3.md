# Kafka Datasource Overview

Configuring via Tyk Dashboard
Click the field to attach Kafka DataSource.
In Configure Data Source, choose KAFKA.
Provide:
Data source name
Broker addresses
Topics
Group ID
Client ID
(Optional) Kafka version, balance strategy, field mappings
Click Save.
